Install Operators and Configure Service(s)

In this section, you will be accessing Hub cluster. You will also be installing additional operators and configuring service(s) needed for deploying Infrastructure and Tenant clusters.

Prerequisites to Access the Hub Cluster.

  1. Create a directory as /root/hub/ on hypervisor.

    mkdir /root/hub

  2. Create kubeadmin-password and console-url files in /root/hub/ directory.

    touch /root/hub/kubeadmin-password; touch /root/hub/console-url

  3. Get the kubeconfig file, password for kubeadmin user and web console url from console.redhat.com.

    1. Download the kubeconfig file to your laptop/desktop and then copy to /root/hub directory on hypervisor.

      Sample output
      [root@hypervisor ~]# ls -al hub/
total 28
drwxr-xr-x.  2 root root  4096 Aug 22 15:20 .
dr-xr-x---. 13 root root  4096 Aug 22 15:18 ..
-rw-r--r--.  1 root root    94 Aug 22 15:11 console-url
-rw-r--r--.  1 root root    24 Aug 22 15:11 kubeadmin-password
-rw-r--r--.  1 root root 12127 Aug 22 15:20 kubeconfig

    2. Copy the password for kubeadmin user and paste it in /root/hub/kubeadmin-password file.

    3. Copy the web console url and paste it in /root/hub/console-url file.

  4. Create .kube directory and config file in .kube directory.

    mkdir /root/.kube; touch /root/.kube/config

Access the Hub Cluster via CLI

  1. Download the OpenShift client (oc).

    wget https://mirror.openshift.com/pub/openshift-v4/clients/ocp/latest/openshift-client-linux.tar.gz

  2. Extract the archive.

    tar -xzvf openshift-client-linux.tar.gz

  3. Move oc and kubectl binaries to /usr/local/bin directory.

    mv oc /usr/local/bin/; mv kubectl /usr/local/bin/

  4. Copy /root/hub/kubeconfig file as /root/.kube/config file.

    cp /root/hub/kubeconfig /root/.kube/config

  5. Set kubepass variable as kubeadmin user’s password.

    kubepass=$(cat /root/hub/kubeadmin-password)

  6. Login to Hub cluster with oc login command.

    oc login -u kubeadmin -p $kubepass
    Sample output
    Login successful.

You have access to 79 projects, the list has been suppressed. You can list all projects with 'oc projects'

Using project "default".

  7. Review the nodes.

    oc get nodes
    Sample output
    NAME                  STATUS   ROLES                         AGE   VERSION
hub.lab.example.com   Ready    control-plane,master,worker   10h   v1.29.7+6abe8a1
